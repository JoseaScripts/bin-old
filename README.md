## Instalación

- Clonar el repositorio
- Crear archivo de claves [~/bin/.claves]

### Clonar repositorio

```shell
clone https://github.com/JoseaScripts/bin
```

### Claves

- Editar el archivo `~/bin/claves` completando las claves

> Después de editar cambiar el nombre del archivo

```shell
mv ~/bin/claves ~/bin/.claves.conf
```

- For all the possible languages that support syntax highlithing on GitHub (which is basically all of them), refer <a href="https://github.com/github/linguist/blob/master/lib/linguist/languages.yml" target="_blank">here</a>.
